# techang skill index

This directory contains the installable skills exposed by `jacobleft/techang`.

## Install from this repo

```bash
npx skills add jacobleft/techang
```

List skills only:

```bash
npx skills add jacobleft/techang --list
```

## Layout expectation

Each skill directory contains a `SKILL.md` file as the entrypoint and may also include support files such as:

- `scripts/`
- `references/`
- `examples/`
- `assets/`
